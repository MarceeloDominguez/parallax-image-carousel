## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/parallax-image-carousel.git
```

Go to the project directory

```bash
  cd parallax-image-carousel
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
